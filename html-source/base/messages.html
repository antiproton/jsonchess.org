/request/time

Requests the current server time from the server. 

	
		{
			"topic": "/request/time",
			"data": String
		}

The client sends an arbitrary identifier, which will be sent back with the server's response to allow the client to associate responses with requests.

See /time/:id (server → client).

/request/user

Requests user details from the server.

See /user (server → client).

/seek

Find a seek matching the specified criteria and accept it, or create a seek for other players to accept. 

	
		{
			"topic": "/seek",
			"data": {
				"initialTime": TimePeriod,
				"timeIncrement": TimePeriod,
				"acceptRatingMin": Rating,
				"acceptRatingMax": Rating
			}
		}

See /seek/waiting, /seek/matched, /seek/expired, /seek/error (server → client); SEEK_TIMEOUT.

/seek/cancel

Cancel the current seek.
